### 字符串

- unicode 扩展：codePointAt、fromCodePoint、at、normalize、正则 u 标志
- 方法扩展
  + 重复：repeat()
  + 确认子串:	includes()、startsWith()、endsWith()，均返回布尔值
  + 补全长度:	padStart()、 padEnd()
  + 正则扩展
- 模板字符串
  + 多行
  + Substitution 变量替换 `${ }`
  + 模板标签函数
  + String.raw()

### 对象

- 字面量语法形式扩展
  + 属性与方法的简写形式
  + 动态计算属性 `[]`
- 方法扩展
  + Object.is()
  + Object.assign()
  + Object.getPrototypeOf()
  + Object.setPrototypeOf()
  + Object.keys()
  + Object.values()
  + Object.entries()
- super
  + super 是指向当前函数原型的指针
  + super 只能在简写方法中使用
